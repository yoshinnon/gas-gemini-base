# シナリオ4: キッチンサイエンス & 猫用品在庫管理

## 概要

**目的:** レシートや購入品リストを AI で解析し、食材の物理・化学特性から科学的な調理アドバイスを生成する。あわせて猫用消耗品の在庫管理・残量予測を統合する。

**用途:**
- 食材の水分・タンパク質量から最適なメイラード反応条件を計算したい
- 「来週月曜」のように調理温度・時間を物理モデルから算出したい
- 猫砂・餌の在庫が少なくなったら警告を受け取りたい
- パスタの乳化（エマルション）を科学的に理解したい
- 日本固有の食材（こんにゃく・おから等）を正しく解析させたい

---

## 設計思想: AI と TypeScript の役割分担

| 担当 | 役割 | 理由 |
|------|------|------|
| **Gemini AI** | 食材名の正規化・栄養成分の推定・保存方法の判断 | 食品知識のデータベースとして活用 |
| **TypeScript** | 比熱計算・メイラードスコア・昇温予測・乳化計算 | 物理モデルは数式で正確に実装できる |

AI にパラメータを抽出させ、TypeScript 側で決定論的に計算することで、計算過程が透明で検証可能になります。

## 物理計算の詳細

### 1. 比熱容量 — Choi-Okos 式

食材の熱的特性の根幹となる比熱容量を成分の加重平均で計算します。

```
c_p = (水分率 × 4.182) + (タンパク質率 × 2.008)
    + (脂質率 × 1.984) + (炭水化物率 × 1.549)
    + (灰分率 × 1.093)   [J/(g·K)]
```

参照: Choi & Okos (1986), "Effects of temperature and composition on the thermal properties of foods"

**例:**
- 鶏むね肉（水分73%, タンパク23.3%）→ 約 3.6 J/(g·K)
- こんにゃく（水分97.3%）→ 約 4.1 J/(g·K)（水の比熱 4.182 に近い）

### 2. メイラード反応スコア（0〜100）

```
タンパク質寄与 = min(60, タンパク質率 × 300)
低水分寄与    = max(0, 40 - 水分率 × 50)
スコア        = タンパク質寄与 + 低水分寄与
```

スコアが高いほどカリッとした焼き色（メイラード反応）が出やすい食材です。

| 食材例 | 水分 | タンパク | スコア |
|--------|------|---------|--------|
| チーズ | 37% | 25.7% | 高 |
| 鶏むね肉 | 73% | 23.3% | 中 |
| こんにゃく | 97% | 0.1% | 低 |

### 3. 表面乾燥時間

```
蒸発エネルギー Q = 表面水分量 [g] × 2257 [J/g（蒸発潜熱@100°C）]
熱フラックス H = 3500 [W] / 314 [cm²] （強火・20cm径パン）
乾燥時間 t = Q / (H × 1 cm²)   [秒]
```

水分量が多い食材（こんにゃく・もやし等）は先に乾煎りするよう指示が追加されます。

### 4. 中心温度到達予測（集中定数系近似）

```
T(t) = T_surface - (T_surface - T_0) × exp(-t / τ)
τ = c_p / (h × A/m)

h = 1000 W/(m²·K)（油を引いたパン面の熱伝達係数）
```

食品安全基準（75°C・1秒以上）の到達時間を厚さから推定します。

> ⚠️ この近似は Biot 数 < 0.1 の場合に有効です。30mm 以上の厚みでは誤差が大きくなります。

### 5. パスタ乳化ガイド（O/W エマルション理論）

```
推奨茹で汁量 = (パスタ重量 / 100g) × 67.5 mL
推奨バター量 = (パスタ重量 / 100g) × 10 g
水分吸収量   = パスタ重量 × min(0.80, 茹で時間秒 / 480 × 0.75)
```

参照: Barham et al. (2010) の推計値をベースにした計算式

### 対応カテゴリー

| カテゴリー | 用途 | デフォルト日消費量（4kg 猫1匹） |
|-----------|------|--------------------------|

| medicine | 薬・外用薬 | 1回/日 |

### アラート閾値

| ステータス | 残り日数 | 表示色 |
|-----------|---------|--------|
| 🟢 余裕あり | 7日超 | 緑（#2e7d32） |
| 🟡 要注意 | 3〜7日 | 黄（#f9a825） |
| 🟠 緊急補充 | 3日以内 | 橙（#ff6d00） |
| 🔴 在庫切れ | 0日 | 赤（#d50000） |

### 拡張性設計

`StockStatus` インターフェースの `healthLog: HealthLogEntry[]` フィールドで、将来的な健康管理ログへの拡張が可能です:

```typescript
interface HealthLogEntry {
  date: string;       // YYYY-MM-DD
  weight?: number;    // 体重 [kg]
  appetite?: "good" | "normal" | "poor";
  note?: string;      // 通院記録・引っかき被害の記録など
}
```


**物理計算のテストカバレッジ:**
- 鶏むね肉の比熱が 2.5〜4.2 J/(g·K) の範囲に収まる
- こんにゃく（水分97%）が水の比熱（4.182）に近い値を持つ
- こんにゃくのメイラードスコアが低い（低タンパク・高水分）
- パスタ重量に比例して茹で汁量が増える
- 厚みが大きいほど75°C 到達時間が長い
- 残り3日以内で critical ステータス
- 在庫0で empty ステータス
- identifier の重複で推奨購入量が正計算される
